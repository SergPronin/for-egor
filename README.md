# Подробное описание программы

Программа представляет собой калькулятор для работы с матрицами с использованием различных методов линейной алгебры. Включает методы для вычисления определителей, решения систем линейных уравнений, нахождения обратных матриц, умножения матриц и приведения матрицы к верхнетреугольному виду.

## Описание классов

1. **`att1`** - Работа с основными операциями над матрицами, такими как вычисление определителей, решение системы уравнений методом Крамера и приведение матрицы к верхнетреугольному виду.
2. **`att2`** - Реализация метода Гаусса для решения систем линейных уравнений.
3. **`att3`** - Операции над матрицами, включая нахождение обратной матрицы для матриц размером 2x2 и произвольного размера.
4. **`MatrixCalculatorUI`** - Графический интерфейс пользователя с использованием Java Swing, позволяющий взаимодействовать с программой.

### Класс `att1`

Этот класс содержит методы для работы с определителями, методом Крамера и приведения матриц к верхнетреугольному виду.

#### 1. `calculateDeterminant(double[][] matrix)`

Метод вычисляет определитель матрицы. Прежде чем вычислить определитель, программа проверяет, является ли матрица квадратной (с количеством строк и столбцов, равным). Если матрица не квадратная, выбрасывается исключение.

**Реализация**:

```java
if (matrix.length != matrix[0].length) {
    throw new IllegalArgumentException("Матрица должна быть квадратной для вычисления определителя");
}
```

Затем вычисляется определитель с помощью рекурсивной функции `computeDeterminant`.

#### 2. `computeDeterminant(double[][] matrix)`

Метод для вычисления определителя матрицы через рекурсивное разложение по строкам (метод Лапласа). Для матриц размером 1x1 и 2x2 используется простой вычислительный алгоритм, а для более крупных матриц вычисление выполняется рекурсивно через минора.

**Реализация**:

```java
if (n == 1) return matrix[0][0];
if (n == 2) return matrix[0][0] * matrix[1][1] - matrix[0][1] * matrix[1][0];
```

Для более крупных матриц используется рекурсивный расчет миноров через метод `computeCofactor`.

#### 3. `computeCofactor(double[][] matrix, int row, int col)`

Этот метод вычисляет кофактор для элемента матрицы, который используется в процессе вычисления определителя.

**Реализация**:

```java
return Math.pow(-1, row + col) * computeDeterminant(getMinor(matrix, row, col));
```

#### 4. `getMinor(double[][] matrix, int row, int col)`

Метод для извлечения минорной матрицы путем удаления строки и столбца, которые содержат указанный элемент.

**Реализация**:

```java
for (int i = 0, r = 0; i < n; i++) {
    if (i == row) continue;
    for (int j = 0, c = 0; j < n; j++) {
        if (j == col) continue;
        minor[r][c++] = matrix[i][j];
    }
    r++;
}
```

#### 5. `solveByCramer(double[][] A, double[][] B)`

Метод для решения системы линейных уравнений методом Крамера. Для этого вычисляется определитель матрицы системы и отдельные детерминанты для каждой переменной (замена столбцов матрицы на вектор правых частей).

**Реализация**:

```java
for (int i = 0; i < A.length; i++) {
    for (int r = 0; r < A.length; r++) {
        temp[r][i] = B[r][0];  // Замена столбца
    }
    result.append("x").append(i + 1).append(" = ").append(computeDeterminant(temp) / detA).append("\n");
}
```

#### 6. `toTriangularForm(double[][] matrix)`

Метод для приведения матрицы к верхнетреугольному виду с помощью метода Гаусса. Во время этого процесса также вычисляется определитель.

**Реализация**:

```java
double detSign = makeUpperTriangular(upper);
double det = detSign;
```

#### 7. `makeUpperTriangular(double[][] matrix)`

Метод для преобразования матрицы в верхнетреугольный вид с помощью метода Гаусса. В процессе также меняются строки матрицы для обеспечения числовой стабильности.

**Реализация**:

```java
if (Math.abs(matrix[k][k]) < EPSILON) {
    for (int i = k + 1; i < n; i++) {
        if (Math.abs(matrix[i][k]) > EPSILON) {
            swapRows(matrix, k, i);  // Меняем строки местами
            detSign *= -1;
            break;
        }
    }
}
```

### Класс `att2`

Этот класс реализует метод Гаусса для решения систем линейных уравнений.

#### 1. `solveByGauss(double[][] A, double[][] B)`

Метод для решения системы линейных уравнений методом Гаусса. Он использует расширенную матрицу и выполняет операции с элементами для получения решения. Для каждого столбца определяется ведущий элемент, и остальные элементы обнуляются.

**Реализация**:

```java
while (pivotRow < rows && Math.abs(matrix[pivotRow][col]) < EPSILON) pivotRow++;  // Поиск ведущего элемента
```

После выполнения метода Гаусса, система либо имеет одно решение, либо не имеет решений, либо имеет бесконечно много решений.

### Класс `att3`

Этот класс содержит методы для работы с матрицами, включая нахождение обратных матриц и умножение матриц.

#### 1. `calculateInverse2x2(double[][] matrix)`

Метод для нахождения обратной матрицы для матрицы размером 2x2. Для этого используется стандартная формула для обратной матрицы 2x2.

**Реализация**:

```java
double det = matrix[0][0] * matrix[1][1] - matrix[0][1] * matrix[1][0];
if (Math.abs(det) < EPSILON) {
    throw new IllegalArgumentException("Определитель равен нулю, обратной матрицы не существует");
}
```

#### 2. `multiplyMatricesToString(double[][] A, double[][] B)`

Метод для умножения двух матриц. Если количество столбцов первой матрицы не совпадает с количеством строк второй, выбрасывается исключение.

**Реализация**:

```java
for (int i = 0; i < A.length; i++) {
    for (int j = 0; j < B[0].length; j++) {
        for (int k = 0; k < A[0].length; k++) {
            result[i][j] += A[i][k] * B[k][j];
        }
    }
}
```

#### 3. `calculateInverseAnySize(double[][] matrix)`

Метод для нахождения обратной матрицы произвольного размера с помощью метода Гаусса-Жордана. Этот метод преобразует матрицу в единичную, а вторую половину расширенной матрицы получает в виде обратной матрицы.

**Реализация**:

```java
for (int i = 0; i < n; i++) {
    if (Math.abs(augmented[i][i]) < EPSILON) {
        for (int k = i + 1; k < n; k++) {
            if (Math.abs(augmented[k][i]) > EPSILON) {
                swapRows(augmented, i, k);
                break;
            }
        }
    }
}
```

### Класс `MatrixCalculatorUI`

Этот класс создает графический интерфейс для взаимодействия с пользователем.

#### Основные элементы:

* **JTable** для ввода матриц A и B.
* **JTextArea** для вывода результатов операций.
* **JButton** для выполнения операций (например, вычисление определителя или решение методом Гаусса).
* Методы для обработки ввода и выполнения операций с матрицами.

**Пример кнопки для вычисления определителя**:

```java
operationPanel.add(createButton("Вычислить определитель", e -> computeDeterminant()));
```

Метод **`getMatrixFromTable()`** извлекает данные из таблицы и преобразует их в массив.

## Заключение

Эта программа реализует множество стандартных операций для работы с матрицами. Она включает в себя как алгоритмические методы, такие как метод Гаусса и метод Крамера, так и математические операции, такие как вычисление определителя, нахождение обратных матриц и приведение матрицы к верхнетреугольному виду. Визуальная часть программы представлена с помощью интерфейса на Swing, что позволяет пользователю удобно вводить данные и получать результаты.
